3 ## リストア restore：ソースコード（intr.S）
4 
5 restore()は、引数（r0）に割り込みマスク値（過去に保存していた値）を取る。
6 ソースコード中に登場する直値 0x01F00220 は、2 進数で表すと<br>
7 0x01F00220 = 0b00000001111100000000001000100000<br>
8 となる。この値は、ビット値の抽出（AND 演算）で用いる。
9 
10 ```
11 restore:
12  push {r1, r2} /* r1, r2をスタックに保存する(リストア中にr1, r2を使用するため) */
13  mrs r1, cpsr　　　　/* cpsrの内容をr1をコピーする */
14  ldr r2, =0x01F00220
15  and r1, r1, r2 /* cpsr(r1)とr2(0x01F00220)の論理積で、関係するフラグをr1に抽出する */
16  bic r0, r0, r2 /* r2を反転したBit値で、割り込みマスク値(引数)の無関係なBitをクリア */
17  orr r1, r1, r0 /* cpsr(コピー)と割り込みマスク(引数)の論理和を取る */
18  msr cpsr_cfsx, r1 /* 割り込みマスク適用済みのcpsrをリストア */
19  pop {r1, r2} /* r1, r2をスタックから復元する */
20  mov pc, lr /* restore()の呼び出し元に戻る */
21 ```
