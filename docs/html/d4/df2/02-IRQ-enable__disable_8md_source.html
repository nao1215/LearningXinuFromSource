11 ## 割り込み禁止 disable：ソースコード（intr.S）
12 
13 C 言語上は、割り込みマスクビット(cpsr)が disable()の返り値(r0)として渡される。<br>
14 この返り値は、 resume()や suspend()の復帰時に cpsr の状態を元に戻すために使用する。
15 
16 ```
17 disable:
18  mrs r0, cpsr /* CPSRレジスタの内容をr0にコピー */
19  cpsid i /* 割り込み禁止状態に遷移 */
20  mov pc, lr /* diableを呼び出す前のアドレスに戻る */
21 ```
